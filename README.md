# Prior Authorization OCR Pipeline

AI-powered prior authorization document extraction system.

## Features
- Azure Document Intelligence OCR
- Structured extractors (Forms, Labs, Imaging, Notes)
- LLM-based clinical summarization
- Deep merge aggregation
- Pydantic schema validation

## Setup

1. Create virtual environment
2. Install requirements
3. Add Azure credentials to .env
4. Run extractors
5. Run aggregate.py
